# Hablai
>## Sistema de Ouvidoria para registro e gerenciamento de reclamações

---
## Funcionalidades: 
1.Listar reclamações registradas;<br>
2.Registrar uma nova reclamação;<br>
3.Pesquisar uma reclamação pelo código;<br>
4.Atualizar uma reclamação existente;<br>
5.Remover uma reclamação pelo código;<br>
6.Mostrar a quantidade total de reclamações cadastradas;<br>
7.Opção para sair do sistema.
---


## Métodos de **python** que serão utilizadas :
- Listas para armazenar as reclamações;<br>
- Laços de repetição como **for** e **while**;<br>
- A função **range()** para auxiliar em percursos e contagens;<br>
- Estrutura de **menu interativo** para o usuário escolher as opções do sistema.<br>
- **Biblioteca Numpy** para listar as reclamações em tabelas
